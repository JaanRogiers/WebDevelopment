Speciale karakters toepassen

We gebruiken tags en attributen om een webpagine op te maken.
Tussen de opentag <h1> en sluittag </h1> plaatsen we hoofdtitels.
Dit kan gaan van <h1> tot <h6>.

Sematische opmaak

We sommen ook nog sommige sematische opmaak tags op:
Met de tag <strong> kun je tekst belangerijk maken.
Met de tag <em> kun je tekst benadrukken.
Met de tag <kbd> toon je keyboard input.
Met de tag <code> toon je stukken computer code.
Met de tag <samp> toon je een output voorbeeld van een computer.
